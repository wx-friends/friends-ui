# Friends 社交交友应用

这是一个基于智能匹配算法的社交交友应用，具有完整的页面交互功能。

## 项目结构

```
Friends/
├── html/
│   ├── js/
│   │   └── navigation.js          # 导航和交互功能脚本
│   ├── home.html                 # 发现页面（主页）
│   ├── chat.html                 # 聊天列表页面
│   ├── moments.html              # 动态广场页面
│   ├── profile.html              # 个人中心页面
│   ├── test.html                 # 功能测试页面
│   └── index.html                # 登录页面
├── PRD.txt                       # 产品需求文档
├── 消息聊天功能实现方案.txt        # 聊天功能实现方案
└── README.md                     # 项目说明文档
```

## 功能特性

### 🎯 核心功能
- **智能匹配系统**：基于用户偏好和兴趣的智能推荐
- **实时聊天**：支持文字、图片、语音消息
- **动态广场**：分享生活动态，互动点赞评论
- **个人中心**：完整的用户资料管理

### 🚀 交互功能
- **页面导航**：统一的底部导航栏，支持页面间切换
- **匹配操作**：喜欢、跳过、超级喜欢功能
- **动态互动**：点赞、评论、分享功能
- **聊天功能**：点击聊天项打开对话
- **个人设置**：编辑资料、会员中心、隐私设置等

### 💡 技术特性
- **响应式设计**：适配移动端和桌面端
- **现代化UI**：使用Tailwind CSS构建的美观界面
- **流畅动画**：丰富的交互动画效果
- **模块化代码**：清晰的代码结构，易于维护

## 页面说明

### 1. 发现页面 (home.html)
- 用户卡片展示
- 匹配操作按钮（喜欢、跳过、超级喜欢）
- 今日匹配次数显示
- 用户信息展示（年龄、距离、职业、兴趣标签）

### 2. 聊天页面 (chat.html)
- 聊天列表展示
- 未读消息提醒
- 在线状态显示
- 点击进入聊天详情

### 3. 动态广场 (moments.html)
- 用户动态展示
- 点赞、评论、分享功能
- 图片展示
- 距离和时间信息

### 4. 个人中心 (profile.html)
- 个人信息展示
- 统计数据（关注、粉丝、动态）
- 功能菜单（编辑资料、会员中心等）
- 认证状态显示

### 5. 功能测试页面 (test.html)
- 所有交互功能的测试
- 导航功能测试
- 提示消息测试
- 功能状态显示

## 使用方法

### 1. 启动项目
直接在浏览器中打开 `html/index.html` 开始使用。

### 2. 页面导航
- 使用底部导航栏在四个主要页面间切换
- 发现页面：浏览和匹配用户
- 消息页面：查看聊天列表
- 动态页面：浏览和互动动态
- 个人中心：管理个人信息和设置

### 3. 交互操作
- **匹配操作**：在发现页面使用喜欢、跳过、超级喜欢按钮
- **聊天功能**：点击聊天列表项进入对话
- **动态互动**：在动态页面点赞、评论、分享
- **个人设置**：在个人中心点击菜单项进行设置

## 技术实现

### 前端技术栈
- **HTML5**：页面结构
- **CSS3**：样式设计（Tailwind CSS）
- **JavaScript**：交互逻辑
- **Font Awesome**：图标库

### 核心功能实现
- **导航系统**：`navigation.js` 提供统一的页面导航
- **交互功能**：各页面独立的交互脚本
- **提示系统**：Toast 消息提示
- **状态管理**：本地状态管理


## 开发说明

### 添加新功能
1. 在 `navigation.js` 中添加新的交互逻辑
2. 在对应页面添加HTML元素和事件监听
3. 更新测试页面验证功能

### 自定义样式
- 修改各页面的CSS样式
- 调整Tailwind CSS类名
- 添加自定义动画效果

### 数据集成
- 替换模拟数据为真实API调用
- 实现用户认证系统
- 添加数据持久化
